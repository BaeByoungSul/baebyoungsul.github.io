3. 폴더 추가
   > assets : 메모 파일 복사
   > environments: environment.ts environment.prod.ts추가
       >> firebase  web app's Firebase configuration 저장
   > core, feature

4. app.config.ts 파일 편집 : 해당라인 추가
   > provideFirebaseApp(() => initializeApp(environment.firebase)),
   > provideDatabase(() => getDatabase()), // Realtime Database 모듈 제공
   > provideAuth(() => getAuth())

5. angular.json 파일 편집
   > 폴더만 추가 하니까 assets에 있는 파일을 HttpClient가 읽지 못한다.
   > assets :  projects > bbs-memo > architect > build > assets 
      >>  ["src/favicon.ico",   "src/assets" ]  
   > environment :  projects > bbs-memo > architect > build > configurations  
      >> budgets 항목 위에 fileReplacements 항목 추가
